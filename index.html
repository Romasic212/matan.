Тригонометрия

Комплексные числа

1) Определение комплексного числа

Комплексное число — это число, которое может быть представлено в виде z = a + bi, где a и b — действительные числа, i — мнимая единица, удовлетворяющая условию i² = -1. Здесь a называется действительной частью комплексного числа, а b — мнимой частью. Комплексные числа позволяют решать уравнения, которые не имеют решений в области действительных чисел, например, уравнение x² + 1 = 0.

2) Алгебраическая форма записи комплексного числа

Алгебраическая форма записи комплексного числа выглядит как z = a + bi, где:

  • a — действительная часть,

  • b — мнимая часть.

Пример: z₁ = 3 + 4i (где a = 3, b = 4).

Сопряженные комплексные числа:

Сопряженное комплексное число к z = a + bi обозначается как ‾z и равно ‾z = a - bi.

Пример: Для z = 3 + 4i, сопряженное число будет ‾z = 3 - 4i.

3) Действия с комплексными числами в алгебраической форме записи
  • Сложение:
    z₁ + z₂ = (a₁ + b₁ i) + (a₂ + b₂ i) = (a₁ + a₂) + (b₁ + b₂)i
  • Вычитание:
    z₁ - z₂ = (a₁ + b₁ i) - (a₂ + b₂ i) = (a₁ - a₂) + (b₁ - b₂)i
  • Умножение:
    z₁ ⋅ z₂ = (a₁ + b₁ i)(a₂ + b₂ i) = (a₁ a₂ - b₁ b₂) + (a₁ b₂ + a₂ b₁)i
  • Деление:
    z₁ / z₂ = (a₁ + b₁ i) / (a₂ + b₂ i) ⋅ (a₂ - b₂ i) / (a₂ - b₂ i) = (a₁ a₂ + b₁ b₂) + (b₁ a₂ - a₁ b₂)i / a₂² + b₂²
4) Модуль и аргумент комплексного числа
  • Модуль комплексного числа z = a + bi определяется как:
    |z| = √(a² + b²)
  • Аргумент комплексного числа:
    Аргумент комплексного числа — это угол θ, который образует вектор, соответствующий комплексному числу, с положительной осью абсцисс. Он может быть найден по формуле:
    θ = tan⁻¹((b / a))

Геометрическое изображение: Комплексное число можно представить на комплексной плоскости (плоскости Оxy), где ось абсцисс соответствует действительной части, а ось ординат — мнимой части.

5) Натуральная степень числа i
  • i⁰ = 1
  • i¹ = i
  • i² = -1
  • i³ = -i
  • i⁴ = 1

Периодичность: Степени числа i повторяются каждые 4: iⁿ = i^(n (mod 4)).

6) Решение квадратных уравнений с отрицательным дискриминантом

Квадратное уравнение имеет вид: ax² + bx + c = 0

Дискриминант определяется как: D = b² - 4ac

Если D < 0, то уравнение имеет два комплексных корня:

x_(1,2) = -b ± √(D) / 2a = -b ± i√(-D) / 2a

Пример:

Решим уравнение x² + 4x + 5 = 0:

  • Здесь a = 1, b = 4, c = 5.
  • Дискриминант: D = 4² - 4*1*5 = 16 - 20 = -4 < 0.
  • Корни: x_(1,2) = -4 ± i√(4) / 2 = -2 ± i.
7) Тригонометрическая форма записи комплексного числа

Тригонометрическая форма записи комплексного числа выглядит так: z = r(cos(θ) + isin(θ)) где:

  • r = |z| = √(a² + b²) — модуль,
  • θ = arg(z) = tan⁻¹((b/a)) — аргумент.

Действия с комплексными числами в тригонометрической форме:

  • Сложение: Необходимо преобразовать в алгебраическую форму.
  • Умножение:
    z₁ z₂ = r₁ r₂ (cos(θ₁ + θ₂) + isin(θ₁ + θ₂))
  • Деление:
    z₁ / z₂ = r₁ / r₂ (cos(θ₁ - θ₂) + isin(θ₁ - θ₂))
8) Показательная форма записи комплексного числа

Показательная форма записи комплексного числа выражается как: z = reⁱᶿ где:

  • r = |z|,
  • θ = arg(z).

Действия в показательной форме:

  • Сложение: Необходимо преобразовать в алгебраическую форму.
  • Умножение:
    z₁ z₂ = r₁ r₂ e^(i(θ₁ + θ₂))
  • Деление:
    z₁ / z₂ = r₁ / r₂ e^(i(θ₁ - θ₂))
